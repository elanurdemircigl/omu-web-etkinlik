# OMÜ Etkinlik Başvuru Formu

Veri gönderimi için **Dummy Endpoint'e POST** tercih ettim. POST metodunu ve webhook.site endpoint'ini kullandım.  
Bu sayede form verilerinin yapılandırılmış bir formatta (JSON) sunucuya ulaşmasını sağlayarak gerçek bir web uygulaması mimarisini simüle ettim.

## Kullanılan Teknolojiler

- **HTML5:** Sayfa yapısı ve form bileşenleri.
- **Webhook.site:** Form verilerinin test edilmesi ve JSON çıktılarının analizi.
